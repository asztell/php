Ken's Main Page

File: kenMain.html

Purpose: to illustrate how the XCMS.php script can display a page specified by the content element in an XML document.

 

You should run the XCMS.php script as follows:

XCMS.php?theXML=kenMain.xml

 

In the kenMain.xml document, the content element is

<content>kenMain.html</content>

Thus, the XCMS.php script displays the kenMain.html file.

 

Style

The style XCMS will use for this page is whatever is defined for the item class in the stylesheet.

The CSS stylesheet to be used is specified by the css element in the kenMain.xml file.

In the kenMain.xml document, the css element is

<css>menuLeft.css</css>
